## The Nine-Sentence Brief

| Decision | Answer |
|---|---|
| **Feeling** | I want this developer like a baby wants candy — immediate, unthinking craving |
| **Audience** | The technical founder who hires himself — burned before, needs fullstack ownership |
| **Anti-audience** | Influencers — no fluff, no vanity, no soft storytelling |
| **Hero object** | Architecture diagram + code snippet — splits apart, seals back together |
| **Job** | Qualify |
| **Cut** | Everything. Only Hero, "How I Build," Projects, GitHub CTA survive |
| **Three-second memory** | "Was he checking me or was I checking the site?" |
| **Color logic** | Warm dark canvas (amber-tinged near-black). Cold technical accent on proof elements only |
| **Type logic** | Geometric display starts restrained, strikes at punishing scale. Monospace for all proof |
| **Spatial logic** | Hero splits apart then seals — single corridor below, one truth per screen |

## Reference Buckets

### Feeling
- **Johan Liebert (Monster)** — Stillness as dominance. The most powerful element does the least.
- **Aizen (Bleach)** — The interaction was scripted before you arrived. Viewer is always one step behind.
- **Warm AMD workstation trap room** — Warmth as a lure. Inviting space contains the most dangerous hardware.
- **Programmer entering the arena (anime)** — Visible competition, invisible hierarchy. Everyone thinks they're the judge.

### Structure
- **linear.app** — One truth per screen. Nothing competes. Confidence is in what's removed.
- **vercel.com** — Code before copy. Proof arrives before the pitch. Technical visitor respected first.

### Detail
- **linear.app** — Weighted scroll reveals. Heavy, not bouncy. Like a vault door opening.
- **vercel.com** — Every interaction intentional. Confident, never decorative.

## Surviving Sections (in order)

1. **Hero** — full viewport, split/seal animation, qualifying statement
2. **How I Build** — architecture thinking, real decisions, full width single column
3. **Projects** — 2–3 max, deep not wide, stack + decision made + GitHub link
4. **GitHub CTA** — one line, one magnetic button, the only door out

**Everything else is cut:** About, Skills grid, Experience timeline, Contact form, social links.

---

## Hero Animation Spec (GSAP)

1. Both panels fade in (0.6s, power2.out)
2. Pull apart — left panel translateX(-8vw), right panel translateX(+8vw) (0.8s, power3.inOut)
3. Hold split for 0.4s
4. Seal back to center (0.7s, power3.inOut)
5. Qualifying statement fades in at punishing scale over the seam (0.5s)

---

## Copy Direction

- Hero statement: max 4 words — appears as the split seals
- Tone: the site doesn't pitch, it filters. Write like someone who already knows the outcome.
- Banned words: modern, clean, minimal, premium, professional, elegant, passionate, hardworking
- CTA: not "contact me" — implies selectivity
